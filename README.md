# 🧠 NLP-Driven Personal Memory Assistant with Real-Time Alerts

## 📌 Overview

This project is an intelligent memory assistant that helps users store, manage, and retrieve important information using Natural Language Processing (NLP). It also provides real-time alerts and notifications based on stored data.

---

## 🚀 Features

* 🔐 User Authentication (Login/Register)
* 🧠 NLP-based memory storage and retrieval
* ⏰ Real-time reminders and alerts
* 📧 Email notifications using Nodemailer
* 💾 MySQL database integration
* 🔄 Real-time updates using backend APIs

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **Other Tools:** Nodemailer, dotenv

---

## ⚙️ Installation & Setup

### 1. Clone the repository

git clone https://github.com/your-username/repo-name.git

### 2. Navigate to project folder

cd project-folder

### 3. Install dependencies

npm install

### 4. Create `.env` file

Add the following:

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=memory_assistant

EMAIL_USER=[your_email@gmail.com](mailto:your_email@gmail.com)
EMAIL_PASS=your_app_password
EMAIL_TO=[receiver_email@gmail.com](mailto:receiver_email@gmail.com)

### 5. Run the project

npm start

---

## 📊 Project Workflow

1. User logs in
2. Stores memory using natural language
3. Data saved in MySQL database
4. System processes input using NLP
5. Alerts/reminders triggered
6. Email notifications sent

---

## 🔐 Security Measures

* Sensitive data stored in `.env` file
* `.env` ignored using `.gitignore`
* Credentials not exposed in code
